Приоритет по желанию:

1. ~~**Профиль**~~ — готово (свой профиль; профиль друга по userId — переход из списка друзей в п.2).

3. **Альбомы** — фотоальбомы пользователя и групп, в т.ч. **сохранённые фото** (photos.get, photos.getAlbums, saved).

5. **Сообщения** — в последнюю очередь (messages.get, messages.getConversations; сложнее по API и модерации VK).

**Улучшения ленты (в бэклоге, приоритет по желанию):**

- **Видео в ленте** — **пока не подтягивается.** Сейчас отображается текст «Видео/ссылка». В бэклоге: превью/плейер в ячейке, просмотр (AVPlayer/video.get). Сложность: средняя.
- **Фото на весь экран** — ✅ сделано (тап по фото поста/альбома → fullscreen + листание).
- **Лайки и комментарии** — в бэклоге: блок под постом (счётчики лайков/комментариев, опционально переход к списку). API: newsfeed.get с полями likes/comments или likes.getList, wall.getComments. То же для фото/видео при открытии.

Цель — «всё необходимое для сервинга ВК»: лента ✅, профиль ✅, дальше друзья, альбомы, группы, затем сообщения; плюс улучшения ленты (видео, полноэкран фото, лайки/комментарии).

## 4. Технические заметки

- **Картинки в постах:** из `attachments[].photo.sizes` берётся URL (приоритет x → m → s), отображается в `LazyVGrid` (1 фото — во всю ширину, 2 — два столбца, 3+ — до 3 столбцов).
- **Пагинация ленты:** при достижении конца списка вызывается `newsfeed.get(start_from: nextFrom)`, посты и авторы мержатся в текущие массивы.
- **Профиль:** `users.get` с fields photo_200, photo_400, photo_max, photo_max_orig. Fullscreen аватара: лучший из доступных (качество зависит от VK). Вкладки: **разбор ошибки VK** — при ответе `{ "error": { "error_code", "error_msg" } }` показывается текст ошибки (VKApiError.apiError → LocalizedError). friends.get extended=1, photos.getAlbums need_covers=1.
- **OAuth scope:** `wall,offline,friends,photos,groups` — после изменения scope нужен **перелогин** (Выйти → Войти), чтобы VK запросил новые права и вкладки Фото/Друзья/Группы получили данные.
- **Фото в ленте:** `displayURL` использует приоритет размеров x,m,s,w,z,y,r,q,p,o → первый доступный; при ошибке загрузки в ячейке показывается иконка. **Часть картинок не подгружается** — открытая проблема (формат sizes/приватность/сеть).
