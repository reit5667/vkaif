## 4. Технические заметки

- **Картинки в постах:** из `attachments[].photo.sizes` берётся URL (приоритет x → m → s), отображается в `LazyVGrid` (1 фото — во всю ширину, 2 — два столбца, 3+ — до 3 столбцов).
- **Пагинация ленты:** при достижении конца списка вызывается `newsfeed.get(start_from: nextFrom)`, посты и авторы мержатся в текущие массивы.
- **Лайки/комментарии в постах:** Под постом: лайк (toggle likes.add/likes.delete, overrides счётчика), комментарии (счётчик, тап → PostCommentsView). PostCommentsView: wall.getComments, пагинация 5, «Добавить комментарий» (корневой), ответ (wall.createComment, reply_to_comment), тап по имени автора → ProfileViewWrapper (programmatic authorDestination), лайк/снять лайк комментария, thread (VKComment.thread), «Подгрузить ещё» скрывается при 0 (noMoreTopLevel). Ответ wall.createComment: декодирование VKWallCreateCommentResponse (comment_id). Кнопка комментариев показывается на всех постах (в т.ч. 0 комментариев).
- **Переход из ленты на группу/профиль:** FeedDestination → GroupWallView или ProfileViewWrapper(userId). Таб «Профиль»: ProfileView(authService, viewModel: profileViewModel) с одним @StateObject profileViewModel в ContentView.
- **Профиль:** users.get, header ок. Вкладки Фото/Друзья/Группы: List вне ScrollView. Друзья: getFriends(count: 5000). Группы: getGroups без filter, цикл по offset; NavigationLink(value: GroupDestination) → GroupWallView. Fullscreen фото: кнопка закрытия слева, тап → панель (счётчики + активные «Нравится»/«Комментарии»); комментарии — sheet из FullScreenPhotoGalleryView (postCommentsContext/photoCommentsContext + authService), открывается поверх fullscreen, «Назад» возвращает на картинку; меню 3 точки, стрелки влево/вправо, свайп вниз.
- **Альбом:** photos.get по 50 (extended=1 для likes/comments), подгрузка при скролле вниз; toolbar — меню 3 точки, сортировка по дате. Fullscreen: лайк (likes.add/delete type photo), комментарии (photos.getComments → PhotoCommentsView), sheet из галереи. **Первый тап по фото:** открывается нужная картинка (fullScreenCover(item: AlbumFullScreenItem(index:))). **Скролл после подгрузки:** внизу LazyVGrid — spacer с высотой под ещё не подгруженные фото (albumBottomSpacerHeight), чтобы контент ScrollView имел запас и скролл опускался; триггер onAppear вызывает loadMoreIfNeeded().
- **OAuth scope:** `wall,offline,friends,photos,groups` — после изменения scope нужен **перелогин** (Выйти → Войти).
- **Фото в ленте:** превью — `feedPreviewURL ?? displayURL`; fullscreen — `displayURL`. В альбомах — `thumbnailDisplayURL ?? displayURL`. **VKPhoto:** поддержка legacy (photo_75, photo_130, photo_604, …); **VKPhotoSize:** url опциональный, поддержка поля "src". Часть превью не подгружается — в бэклоге (точная причина не выявлена).
