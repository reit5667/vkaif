## 2. Дорожная карта (дальше)

Приоритет по желанию:

1. ~~**Профиль**~~ — готово (свой профиль; профиль друга по userId — переход из списка друзей в п.2).
2. **Друзья** — список друзей (friends.get), переход в профиль друга.
3. **Альбомы** — фотоальбомы пользователя и групп, в т.ч. **сохранённые фото** (photos.get, photos.getAlbums, saved).
4. **Группы** — список групп пользователя, страница группы (groups.get, wall группы).
5. **Сообщения** — в последнюю очередь (messages.get, messages.getConversations; сложнее по API и модерации VK).

**Улучшения ленты (в статус, приоритет по желанию):**

- **Видео в ленте** — сейчас не отображаются. Нужно: превью/плейер в ячейке поста, просмотр (AVPlayer/встроенный плеер). Сложность: средняя — VK отдаёт video в attachments, для воспроизведения нужен video.get или прямые ссылки на файлы; возможны ограничения по встраиванию.
- **Фото на весь экран** — раскрытие картинки поста на весь экран + листание в пределах поста (TabView/PageView по фото одного поста).
- **Лайки и комментарии** — блок под постом: счётчики и (опционально) переход к списку лайков/комментариев. API: likes.getList, wall.getComments (или из полей поста likes.count, comments.count при расширении newsfeed.get полями).

Цель — «всё необходимое для сервинга ВК»: лента ✅, профиль ✅, дальше друзья, альбомы, группы, затем сообщения; плюс улучшения ленты (видео, полноэкран фото, лайки/комментарии).

## 4. Технические заметки

- **Картинки в постах:** из `attachments[].photo.sizes` берётся URL (приоритет x → m → s), отображается в `LazyVGrid` (1 фото — во всю ширину, 2 — два столбца, 3+ — до 3 столбцов).
- **Пагинация ленты:** при достижении конца списка вызывается `newsfeed.get(start_from: nextFrom)`, посты и авторы мержатся в текущие массивы.
- **Профиль:** `users.get` без user_ids = текущий пользователь; с `user_ids` = профиль друга. Модель `VKUserDetail` (id, firstName, lastName, photo50, photo200, status). Экран `ProfileView` — аватар, имя, статус; переход в профиль друга по `userId` (будет использоваться из списка друзей).
