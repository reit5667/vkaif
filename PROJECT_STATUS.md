## Вводные по работе с проектом

- **Не запускать сборку/тесты из терминала** (xcodebuild, swift build и т.п.). Сборку и проверку делает пользователь в Xcode; в диалогах не пытаться «протестировать» через команды — это не требуется и не используется. В Xcode работа в той же папке - изменения подтягиваются автоматически.
- По необходимости подтягивать контекст из **PROJECT_STATUS.md**, **TECHNICAL_NOTES.md**, **MESSAGES_SPEC.md**.

## 1. Что уже есть (кратко)

- **Лента:** посты с фото, видео, ссылками, опросами; лайки, комментарии, репосты fullscreen фото/видео; тап по автору → группа/профиль; меню «три точки» у своих постов (Удалить, Закрепить/Открепить); дата поста (календарная + относительная); удаление фото в fullscreen для своих фото.
- **Профиль:** header (users.get), вкладки Стена / Фото / Друзья / Группы; альбомы, «Сохранённые фото», стена группы, профиля; удалить пост/фото; закрепить/открепить пост на своей стене.
- **Фото в fullscreen:** меню «три точки» — «Добавить в сохранённые» (photos.copy, галерея держит vkApi), «Скачать на устройство» (PHPhotoLibrary, MainActor), «Удалить» для своих фото/постов; в альбомах — то же меню, удаление фото из своих альбомов (photos.delete, при пустом токене из галереи берётся authService.accessToken в deletePhotoFromAlbum).
- **Таббар:** Лента | Друзья | Сообщения | Профиль.
- **Сообщения:** список диалогов (getConversations), чат (getHistory, send), пагинация старых сообщений, отправка фото в диалоге (getMessagesUploadServer → saveMessagesPhoto → send с attachment). Scope OAuth: messages.
- Детали по завершённым фичам — в PROJECT_STATUS_ARCHIVE.md.
