# PRD — VKaif UI Redesign
**Версия:** 1.0  
**Дата:** 2026-06-28  
**Автор:** Vadim Sharavin  
**Тип:** UI/UX редизайн

---

## 1. Обзор и цели

Полный редизайн интерфейса VKaif в стиле классического ВКонтакте эпохи 2012–2016 гг. с адаптацией под современный iOS. Текущий интерфейс использует стандартные SwiftUI-компоненты (серые пузыри, скруглённые элементы, нижняя таб-бар), которые не соответствуют заявленной концепции продукта.

### Цели редизайна
- Передать визуальную эстетику старого VK: насыщенный синий, плоские элементы, строгая типографика
- Убрать нижнюю таб-бар — освободить экран для контента
- Ввести drawer-навигацию по свайпу влево
- Сделать интерфейс контрастным и читаемым (референс: LinkedIn, старый VK)
- Применить единую дизайн-систему ко всем экранам приложения

---

## 2. Целевая аудитория

Та же, что у основного продукта: пользователи ВКонтакте 20–40 лет, помнящие или ценящие классический интерфейс VK. Ностальгия — часть ценностного предложения.

## 3. Дизайн-система

### 3.1 Цветовая палитра

| Роль | Цвет | HEX |
|------|------|-----|
| Primary (шапки, кнопки, акцент) | Классический VK Blue | `#4A76A8` |
| Primary Dark (нажатое состояние) | — | `#3D6491` |
| Background | Белый | `#FFFFFF` |
| Surface (карточки, блоки) | Белый | `#FFFFFF` |
| Separator / Border | Светло-серый | `#E0E0E0` |
| Secondary Background | Очень светло-серый | `#F5F5F5` |
| Text Primary | Тёмно-серый почти чёрный | `#222222` |
| Text Secondary | Средне-серый | `#828282` |
| Text on Blue | Белый | `#FFFFFF` |
| Incoming bubble | Светло-голубой | `#D9E8F5` |
| Outgoing bubble | Белый с рамкой | `#FFFFFF` |
| Drawer background | Тёмно-синий | `#2D4F72` |
| Drawer text | Белый | `#FFFFFF` |
| Badge | Красный | `#E64646` |
| Online indicator | Зелёный | `#4BB34B` |

**Только светлая тема.** Тёмная тема не предусмотрена.

### 3.2 Типографика

Системный шрифт San Francisco (SF Pro). Использовать стандартные размеры iOS без кастомных весов — чистый, прямой стиль.

| Роль | Size | Weight |
|------|------|--------|
| Заголовок экрана (nav bar) | 17pt | Semibold |
| Имя пользователя в посте/диалоге | 15pt | Semibold |
| Текст поста/сообщения | 15pt | Regular |
| Вторичный текст (время, статус) | 13pt | Regular |
| Статистика — цифра | 17pt | Bold |
| Статистика — подпись | 12pt | Regular |
| Секционный заголовок | 13pt | Semibold (UPPERCASE) |

### 3.3 Форма и геометрия

- **Скругление кнопок:** `cornerRadius: 4` — минимальное, почти прямой угол
- **Скругление карточек:** `0` — прямые углы
- **Аватарки в постах и профиле:** квадратные, `cornerRadius: 2`
- **Аватарки в диалогах:** круглые (исторически так в VK-мессенджере)
- **Тени:** отсутствуют везде
- **Градиенты:** отсутствуют везде
- **Разделители:** тонкая линия `1px`, цвет `#E0E0E0`

### 3.4 Анимации и эффекты

- Системные iOS transitions (push, modal) — без кастомных анимаций
- Drawer: стандартный slide-in слева
- Tap feedback: стандартный opacity dimming
- Никаких parallax, blur, bounce-эффектов

## 4. Навигация

### 4.1 Drawer-меню (главная навигация)

**Триггер:** свайп влево от левого края экрана (или хамбургер-иконка в шапке, если применимо)

**Структура drawer:**

```
┌─────────────────────────┐
│  [Аватар] Имя           │  ← Блок профиля, отдельный, кликабельный
│           @screen_name  │
├─────────────────────────┤
│  📰  Лента              │
│  👥  Друзья             │
│  ✉️  Сообщения    [17]  │  ← бейдж непрочитанных
│  👥  Группы             │
│  🔍  Поиск              │
├─────────────────────────┤
│  ⚙️  Настройки          │
└─────────────────────────┘
```

**Визуал:** тёмно-синий фон `#2D4F72`, белые иконки (SF Symbols) + белый текст, бейджи красные. Блок профиля отделён от пунктов меню разделителем.

**Acceptance criteria:**
- Свайп от левого края (>30pt) открывает drawer
- Свайп вправо или тап по затемнённой области — закрывает
- Активный пункт меню подсвечен (белый фон с opacity или синяя полоска слева)
- Бейдж на "Сообщения" обновляется в реальном времени

### 4.2 Отмена нижней таб-бар

`TabView` полностью убирается. Всё приложение строится на NavigationStack + Drawer.

## 5. Экраны

### 5.1 Лента (Feed)

**Описание:** Стартовый экран приложения. Хронологическая лента постов.

**Шапка:** Синяя `#4A76A8`, белый текст "Новости", иконка хамбургера (drawer) слева с бейджем непрочитанных сообщений. *(решение 2026-06-28: шапка оставлена — удобна для доступа к drawer)*

**"Обновить ленту":** pull-to-refresh (стандартный iOS).

**Карточка поста:**
```
┌─────────────────────────────────────────┐
│ [Аватар 40×40] Имя Фамилия    10:32    │
│                Имя группы (если репост) │
├─────────────────────────────────────────┤
│ Текст поста...                          │
│                                         │
│ [Фото / Видео / Вложение]               │
├─────────────────────────────────────────┤
│ 💬 23    🔁 5    ❤️ 147                │
└─────────────────────────────────────────┘
```

- Фон карточки: белый, граница `1px #E0E0E0`, cornerRadius 0
- Аватар: квадратный `40×40`, cornerRadius 2
- Имя: `15pt Semibold`, цвет `#4A76A8` (кликабельная ссылка)
- Время: `13pt Regular`, цвет `#828282`, выравнивание по правому краю
- Текст: `15pt Regular #222222`
- Панель действий: иконка SF Symbol + цифра, цвет `#828282`; лайк при нажатии — красный

**Acceptance criteria:**
- Бесконечная прокрутка (пагинация VK API)
- Без Stories, без рекламных постов (marked_as_ads отфильтрованы)
- Нет нижней таб-бар — лента занимает весь экран

### 5.2 Сообщения — список диалогов

**Шапка:** Синяя `#4A76A8`, белый текст "Сообщения", иконка карандаша (новое сообщение) справа.

**Ячейка диалога:**
```
┌──────────────────────────────────────────┐
│ [Аватар]  Имя Фамилия / Беседа   16:46  │
│  48×48    Превью последнего сообщ...  🔴 │
└──────────────────────────────────────────┘
```

- Аватар: **круглый** `48×48` (исторически в VK-мессенджере)
- Имя: `15pt Semibold #222222`
- Превью: `13pt Regular #828282`, обрезается в одну строку
- Время: `13pt Regular #828282`
- Бейдж непрочитанных: красный круг, белый текст
- Онлайн-индикатор: зелёная точка поверх аватара (правый нижний угол)
- Разделитель между ячейками: `1px #E0E0E0`

**Acceptance criteria:**
- Список загружается из VK API (conversations)
- Непрочитанные диалоги — жирное имя
- Тап на диалог → открывает чат

### 5.3 Чат

**Шапка:** Синяя, стрелка назад слева, в центре — аватар (круглый, `32px`) + имя + "был(а) в сети HH:MM", справа — иконка скрепки (вложение) + три точки (меню).

**Пузыри сообщений:**

```
Входящие (левый край):
[Аватар] ┌──────────────────┐
         │ Текст сообщения  │  фон #D9E8F5
         │                  │
         └──────────────────┘
                              10:07

Исходящие (правый край):
         ┌──────────────────┐
         │ Текст сообщения  │  фон #FFFFFF, border #E0E0E0
         │                  │
         └──────────────────┘  ✓✓ 10:08
```

- Скругление пузырей: `cornerRadius: 12` (исключение из правила — исторически так в мессенджере)
- Разделитель даты: серый текст по центру "8 февраля"
- Статус доставки: ✓ (доставлено) / ✓✓ (прочитано)

**Инпут-бар (снизу):**
```
┌──────────────────────────────────────────┐
│ 📎  [ Ваше сообщение...    ] 😊  ▶️    │
└──────────────────────────────────────────┘
```
- Фон бара: белый, верхняя граница `1px #E0E0E0`
- Кнопка "Отправить": синяя стрелка, активна только если есть текст

**Acceptance criteria:**
- Отправка текстовых сообщений
- Reply на сообщение (цитата)
- Отображение вложений: фото, стикеры, голосовые, документы
- Статус прочтения
- Бесконечная прокрутка истории вверх

### 5.4 Профиль

**Шапка:** Синяя, текст "Профиль" (или имя пользователя при просмотре чужого).

**Блок шапки профиля:**
```
┌───────────────────────────────────────────┐
│  [Аватар 72×72]  Имя Фамилия ✓           │
│   квадратный     был(а) в сети 22 мин назад│
│                  Статус-текст             │
├───────────────────────────────────────────┤
│  [ Сообщение ]      [ Добавить в друзья ] │  ← плоские кнопки
├───────────────────────────────────────────┤
│  1.1K      3       353      558    6      │
│  друзей  общих  подписч.  фото  видео    │
├───────────────────────────────────────────┤
│  🏠 Город: Москва                        │
│  💼 Место работы: ...                    │
│  🔔 353 подписчика                       │
│  ℹ️  Подробная информация →              │
├───────────────────────────────────────────┤
│  ФОТОГРАФИИ  558                    Все › │
│  [фото] [фото] [фото] [фото]             │
├───────────────────────────────────────────┤
│  ВСЕ ЗАПИСИ    МОИ ЗАПИСИ               │
│  ─────────────                           │  ← синяя подчёркивающая линия
├───────────────────────────────────────────┤
│  [Посты в виде карточек, как в ленте]     │
```

**Детали:**
- Аватар: квадратный `72×72`, cornerRadius 2
- Кнопки действий: плоские, cornerRadius 4, левая — синяя заливка, правая — белая с синей рамкой
- Статистика: `17pt Bold` цифра + `12pt Regular` подпись под ней, горизонтальный ряд с разделителями
- Инфо-блок: иконка SF Symbol (серая) + текст, каждый пункт на отдельной строке
- Фото-стрип: горизонтальный скролл, квадратные превью
- Вкладки: `13pt Semibold UPPERCASE`, активная вкладка — синяя подчёркивающая линия `2px`
- Без обложки (cover photo)

**Acceptance criteria:**
- Корректно отображает свой и чужой профиль
- Статистика кликабельна (переход в раздел)
- Фото-стрип → открывает альбом профиля
- Вкладки переключают ленту записей

## 6. Технические требования

### 6.1 Дизайн-токены (SwiftUI)

Создать файл `VKTheme.swift` с константами:

```swift
// Colors
static let primary = Color(hex: "#4A76A8")
static let primaryDark = Color(hex: "#3D6491")
static let separator = Color(hex: "#E0E0E0")
static let textPrimary = Color(hex: "#222222")
static let textSecondary = Color(hex: "#828282")
static let incomingBubble = Color(hex: "#D9E8F5")
static let drawerBackground = Color(hex: "#2D4F72")
static let badge = Color(hex: "#E64646")
static let online = Color(hex: "#4BB34B")

// Geometry
static let buttonRadius: CGFloat = 4
static let avatarRadius: CGFloat = 2
static let bubbleRadius: CGFloat = 12
static let cardBorderWidth: CGFloat = 1
```

### 6.2 Компоненты (переиспользуемые)

| Компонент | Описание |
|-----------|----------|
| `VKNavigationBar` | Синяя шапка, белый текст/иконки |
| `VKButton` | Плоская кнопка cornerRadius 4, primary/secondary стили |
| `VKAvatar` | Квадратный или круглый аватар с онлайн-индикатором |
| `VKPostCard` | Карточка поста: аватар + шапка + контент + панель действий |
| `VKMessageCell` | Ячейка диалога |
| `VKBubble` | Пузырь сообщения (incoming/outgoing) |
| `VKDrawer` | Drawer-меню со свайпом |
| `VKBadge` | Красный бейдж с цифрой |
| `VKSectionHeader` | Серый заголовок секции UPPERCASE |
| `VKStatGrid` | Сетка статистики профиля |

### 6.3 Платформа

- iOS 16+
- SwiftUI
- Только светлая тема (`.preferredColorScheme(.light)` на уровне приложения)
