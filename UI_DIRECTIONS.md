# VKaif — UI направления редизайна

Референс: VK Android 3.0 (`design_vk/android_30/`)

## Решения по аватаркам (глобальные)

| Место | Форма |
|---|---|
| Список диалогов (MessagesTabView) | Квадратная, cornerRadius 4 |
| Чат — шапка (ChatView toolbar) | Круглая 32px |
| Drawer/Sidebar | Круглая 44px |
| Профиль — шапка | Квадратная ~100pt |
| Страница сообщества | Квадратная |
| Список друзей | Квадратная, cornerRadius 4 |
| Комментарии | Квадратная, cornerRadius 4 |
| Лента — карточка поста | Квадратная 40pt, cornerRadius 2 |

## 1. Профиль + Авторизация
Референс: `login_profile.jpg`

### Авторизация (OnboardingView)
- Голубой фон VKTheme.Colors.primary
- Логотип **VKaif** (не В) крупно по центру
- Поля email/пароль с белым фоном
- Кнопка "Войти" белая с синим текстом
- Ссылка "Зарегистрироваться" внизу белым

### Профиль пользователя (ProfileView)
- Синяя шапка VKNavigationBar "Профиль"
- Квадратная аватарка ~100pt, cornerRadius 2
- Имя 18pt Bold + галочка верификации + стрелка раскрытия инфо
- "Был(а) в сети …" 13pt серым
- Статус-текст пользователя
- Кнопки: "Добавить в друзья" + "Личное сообщение" (для чужого профиля); "Редактировать" (для своего)
- **Блок статистики 2×3** на тёмном фоне `#1F2B38`: друзья / подписчики / фото / видео / аудио / группы — цифры белые Bold, подписи серые 12pt
- Горизонтальный фотострип (5 квадратных превью)
- Вкладки "ВСЕ ЗАПИСИ" / "ЗАПИСИ {ИМЯ}" — 13pt Semibold uppercase, активная — синяя линия 2px
- Лента постов ниже вкладок

### Экран "Информация" (новая задача — нет в tasks.json)
- Открывается по тапу на стрелку в шапке профиля
- День рождения, родной город, семейное положение
- Родственники (аватарка + имя + роль)
- Контактная информация (город, сайт, etc.)

## 2. Сообщения + Лента
Референс: `messages_feed.jpg`

### Список диалогов (MessagesTabView)
- Аватарки квадратные со скруглением cornerRadius 4
- Имя 15pt Semibold + время 13pt серым
- Превью последнего сообщения: если наше — "Вы: " серым + текст; если входящее — просто текст
- Онлайн-точка зелёная (#4BB34B) в правом нижнем углу аватарки
- Непрочитанные: имя Bold + красный бейдж-капсула справа
- Разделители с отступом от аватарки

### Чат (ChatView)
- Синяя шапка: круглая аватарка 32px + имя + "Не в сети / в сети"
- Фон чата `#D5DDE5`
- Входящие пузыри: белый фон, cornerRadius 12
- Исходящие пузыри: голубой фон `#D9E8F5`, cornerRadius 12
- Инпут: иконки вложений | поле "Сообщение..." | кнопка "Отпр." синяя

### Лента (FeedView)
- Синяя шапка "Новости" + иконка карандаша справа
- Баннер новых записей: **синяя текстовая ссылка** "Показать N новых записей" по центру вверху (не таблетка)
- Карточка поста: квадратная аватарка 40pt, имя синим, дата-время серым, текст, сетка фото

---

## 3. Друзья + Фотографии
Референс: `friends_albums.jpg`

### Список друзей (FriendsTabView)
- Вкладки: "N ДРУЗЕЙ" / "N В СЕТИ" / "РЕКОМЕНДАЦИИ" — uppercase, активная синяя линия 2px
- Секция "ВАЖНЫЕ" с синим заголовком
- Алфавитные секции (А, Б, В...)
- Аватарки квадратные cornerRadius 4, имя 15pt, онлайн-точка синяя (не зелёная в этом экране — по референсу)
- Рекомендации: карточки с кнопками "✓ Добавить" (синяя) / "Скрыть" (серая)

### Фотографии (альбомы профиля)
- Вкладки: "ФОТО" (все фото профиля) / "АЛЬБОМЫ"
- Сетка: **2 альбома в ширину** (не 1 на весь экран)
- Обложка альбома: превью с overlay названия и счётчика
- Тёмный фон `#1F2B38` у каждой плитки
- Вкладка "ДРУЗЬЯ" из референса — **не реализуем**, заменяем на "ФОТО"

## 4. Сообщества
Референс: `communities_bookmarks.jpg`

### Список сообществ (ProfileGroupsTabView / GroupsTabView)
- Синяя шапка "Сообщества" + иконка поиска
- Вкладки: "N СООБЩЕСТВ" / "N ВСТРЕЧИ" (встречи пока можно опустить)
- Ячейка: квадратная аватарка, название Bold, тип "Публичная страница" серым

### Страница сообщества (GroupWallView)
- Синяя шапка с названием сообщества
- Квадратная аватарка, название, описание со стрелкой раскрытия
- Статистика (участники / фото / аудио) на тёмном фоне `#1F2B38`
- Горизонтальный фотострип
- Кнопка "Написать на стене" синяя во всю ширину
- Лента постов сообщества

---

## 5. Левый сайдбар + Настройки
Референс: `leftsidebar_setting.jpg`

### DrawerMenuView
- Тёмный фон `#2D4F72`
- Аватарка **круглая** 44px вверху
- Имя пользователя 15pt Semibold + `@id{number}` 12pt серым
- Пункты меню 17pt: Лента / Друзья / Сообщения / Сообщества / Аудио / Настройки
- Иконки серые, активный пункт — белый + подсветка фона
- Бейдж непрочитанных на Сообщениях
- Мини-плеер внизу если играет музыка (уже есть)

### Настройки (SettingsView)
- Синяя шапка "Настройки"
- Секции: ОБЩИЕ / ОПОВЕЩЕНИЯ (пока заглушки)
- Кнопка "Выйти" красного цвета внизу

---

## 6. Комментарии к посту
Нет отдельного референса — стиль должен совпадать с лентой.

### PostCommentsView
- Синяя шапка "Комментарии N"
- Ячейка: квадратная аватарка 36px + имя 14pt Semibold (синее) + текст 14pt Regular (не 17pt!)
- Тап по аватарке → ProfileView
- Тап по имени → ProfileView
- Время 12pt серым справа
- Плоский инпут: поле + кнопка отправки синяя
- Разделители между комментариями
