# VKaif

iOS-клиент ВКонтакте без рекламы, рекомендаций и алгоритмической ленты.

Показывает только то, на что ты подписан — в хронологическом порядке.

---

## Что умеет

- **Лента** — посты без рекламы, репостов от незнакомцев и навязанного контента
- **Чаты** — личные сообщения со стикерами, голосовыми, фото, видео и документами
- **Профиль** — стена, альбомы, смена аватара
- **Друзья** — список друзей с переходом в диалог или профиль
- **Fullscreen-галерея** — pinch-зум, листание, скачивание фото

## Стек

- iOS 16+, SwiftUI, MVVM
- VK API (OAuth через Kate Mobile token)
- AVFoundation (голосовые сообщения)
- StoreKit 2 (In-App Purchase, в разработке)

## Авторизация

Приложение использует токен VK Kate Mobile для доступа к API. Токен вводится вручную на экране авторизации.

В активной разработке. Цель — релиз в App Store.
